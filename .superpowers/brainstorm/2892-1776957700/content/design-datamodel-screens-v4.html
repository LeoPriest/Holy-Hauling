New: User model
-- users table
id             VARCHAR  PRIMARY KEY  (uuid)
username       VARCHAR  NOT NULL UNIQUE
credential_hash VARCHAR  NOT NULL     (bcrypt — PIN today, password later)
role           VARCHAR  NOT NULL     (admin | facilitator | supervisor | crew)
is_active      BOOLEAN  DEFAULT true
created_at     DATETIME NOT NULL
created_by     VARCHAR              (user_id of admin who created)
Seeded at startup: one default admin user with PIN 0000 if the table is empty.
New API routes
POST /auth/login — {username, pin} → {token, user}. No auth required.
GET /auth/me — verify token still valid on app load.
GET /admin/users · POST /admin/users · PATCH /admin/users/{id} — admin only.
GET /jobs — booked leads for crew/supervisor. Auth required.
PATCH /jobs/{id}/status — en_route / started / completed. Supervisor only.
